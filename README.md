## Objetivo do Software
O sistema tem como objetivo gerenciar suas HQs, permitindo adicionar, consultar, editar e remover registros de quadrinhos.

## Funcionalidades do Sistema (Requisitos)
1. Adicionar HQs com informações como título, autor, ano e numeração.
2. Consultar HQs no banco de dados.
3. Editar informações de HQs cadastradas.
4. Remover HQs do sistema.
5. Verificar a existência de um título antes de cadastro.
